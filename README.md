# Python-Practice-02
# 🐍 Python Basics – Even/Odd Checker & Sum of Numbers

This project contains two beginner-friendly Python programs that demonstrate the use of **if-else statements** and **for loops**.

## 📌 Tasks Included

### ✅ Task 1: Check if a Number is Even or Odd

This program:

- Takes an integer input from the user.
- Checks whether the number is even or odd using an `if-else` statement.
- Displays the result.

### Example

**Input**

```
Enter a number: 12
```

**Output**

```
12 is an even number
```

---

### ✅ Task 2: Sum of Integers from 1 to 50

This program:

- Uses a `for` loop to iterate through numbers from **1 to 50**.
- Calculates the sum of all integers in this range.
- Displays the final sum.

### Output

```
The sum of numbers from 1 to 50 is: 1275
```
